Access & Password
How to access your admin panel securely from GitHub Pages.

🔗 Your Admin URL

After pushing to GitHub Pages, your admin is at:

yourusername.github.io/portfolio/admin.html

The main site is still yourusername.github.io/portfolio — they're separate.

✅ Easy to access from any device ⚠️ Anyone who knows the URL can try to access it

🔐 Password Protection

The admin is protected by the password you set below. Change it from the default immediately.

The password is stored in admin.html itself — open the file in a text editor and search for ADMIN_PASSWORD.

✅ Simple, no server needed ✅ Works on GitHub Pages (free)

💻 Local Use (Recommended)

Download admin.html, open it directly in your browser from your computer. No internet needed. No one else can access it.

✅ Most secure option ✅ Works offline ⚠️ Can't access from phone/other device

📁 Private Repo Option

Keep your portfolio public, but move admin.html to a separate private repo. Then just open it locally when needed.

✅ Most private — admin never online ✅ Main site stays fully public

🔑 Change Admin Password

After changing, download data.json AND re-download admin.html (or edit the file manually — search for ADMIN_PASSWORD in the file).
